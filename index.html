Screen Readers

  • Для навигации и поиска по контенту используется клавиатура
  • Простая и функциональная система шорткатов
  • 98.6% пользователей screen reader'ов users не выключают JS!

Screen Reader игнорирует

  • img с пустыми alt аттрибутами alt=""
  • display: none;
  • visibility: hidden;
  • :before content, :after content (тип того)
  • Держите в голове иконки и иконочные шрифты!

Screen Reader не забудет

  • Контент, спрятанный через opacity, z-index, height
  • oﬀ-screen позиционирование (text-indent, top, left)
  • CSS clipping

High Contrast Mode

Кому нужен High Contrast mode?

  • Слабовидяшим пользователям
  • Пользователям с дальтонизмом
  • Пользователям с чувствительным зрением

А что под капотом?

  • Изменяется цветовая гамма (text color, borders, etc.)
  • Убираются фоновые цвета и изображения
  • Также увеличиваются размеры шрифтов

No background???

  • Ииии...

Fix it!

  • У нас есть:
  • HTML: <button type="button">Print</button>
  • CSS: background: url(sprite.png);

Fix it!

			button {
				overflow: hidden;
			}
			button:before {
				content: url(sprite.png);
				margin: -X 0 0 -Y;
			}

High Contrast Mode mini checklist

  • Background-color

  • Colors
  • Transparent borders
  • Transparent Outlines
  • <img> Images
